# E-commerce-Sales-Analytics-System
Analyzed 99k+ e-commerce orders using Python and Pandas. Performed data cleaning, revenue analysis, customer segmentation, seller performance analysis, product category analysis, payment analysis, and geographic sales analysis. Generated business insights through data aggregation, grouping, and time-series analysis.
